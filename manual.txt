Lo primero deben clonar el repositorio de la nuebe a sus PCs:
mejor si lo clonan dentro de la carpeta htdocs de XAMPP para 
asi poderlo correr desde el navegador:

$git clone https:................

Luego Primera configuración anotan su nombre de 
usuario y correo

$ git config --global user.name "suUsuario"
$ git config --global user.email johndoe@example.com

para ver la configuracio que hicieron realizen lo siguiente:

$git config --list

Para entrar al repositorio clonado:

$cd repo162

Una vez dentro de la carpeta repo162 lo q pueden hacer es crear nuevos archivos 
o bien editar archivos ya existentes, cuando hayan terminado de trabajar lo suben
con los siguientes comandos:

$git add *
$git commit -m "Lista la Funcion Fibonacci"
$git push -u origin master

Nota: Si al subir los archivos les sale error es por q quiza no tienen la version mas actualizada del
