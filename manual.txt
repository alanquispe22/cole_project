$git clone https:................

 
Para entrar al repositorio clonado:

$cd repo162

Luego Primera configuración anotan su nombre de 
usuario y correo

$ git config --global user.name "suUsuario"
$ git config --global user.email johndoe@example.com

para ver la configuracio que hicieron realizen lo siguiente:

$git config --list
